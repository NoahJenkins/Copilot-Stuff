# Skills

This directory contains reusable GitHub Copilot skill artifacts.

## Available Skills

### sync-agents

- Description: Syncs GitHub Copilot instructions, custom agents, and skills into Claude Code, Gemini CLI, and OpenAI Codex formats.
- Source: [sync-agents/SKILL.md](./sync-agents/SKILL.md)
- Raw URL: `https://raw.githubusercontent.com/NoahJenkins/Copilot-Stuff/main/skills/sync-agents/SKILL.md`

- Install: Copy the entire `skills/sync-agents/` folder into your workspace skill location (for repo-scoped usage: `.github/skills/sync-agents/`).

## Note

VS Code supports one-click install links for prompts, agents, and instructions. Skills are folder-based artifacts, so the reliable install path is to copy the full skill folder.
