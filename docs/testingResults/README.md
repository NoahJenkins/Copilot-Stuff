# Testing Results

This directory stores dated test feedback reports for artifact validation runs performed against `testing_sandbox/`.

## Naming Convention

- `YYYY-MM-DD-<artifact-name>.md`

Examples:

- `2026-03-12-test-artifact-skill.md`
- `2026-03-12-prompt-engineer-agent.md`

## Required Sections

- Artifact under test
- Date
- Setup performed
- Test actions executed
- Observed behavior
- Pass/fail verdict
- Recommended improvements

Use `.github/skills/test-artifact/SKILL.md` for the full workflow and reporting template.
